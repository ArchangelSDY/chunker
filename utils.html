Utils¶

class utils.OffsetFilePtr(fp, offset, total_length)¶

A wrapper for file object to maintain a sub file in original stream.

Parameters:
  • fp – Original file object.
  • offset – Start position of the sub file.
  • total_length – Length of the sub file.
close()¶

Set internal references to None.

Note that it will not close the original file object.

read(n)¶

Read bytes.

Parameters:n – Count of bytes to read.
Returns:Bytes.
seek(pos, mode)¶

Seek position in sub file.

Parameters:
  • pos – Position.
  • mode – Seek mode.
tell()¶

Current position in sub file.

Returns:Current position.
